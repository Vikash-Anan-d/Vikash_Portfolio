# Vikash Anand - Advanced Developer Portfolio

A highly advanced, modern, and visually stunning personal developer portfolio website built with Next.js App Router, Tailwind CSS, Framer Motion, and GSAP. 

## Features
- **Next.js 14 App Router**: Fast, SEO optimized, and scalable.
- **Tailwind CSS**: Custom dark mode, glassmorphism utilities, and responsive design.
- **Framer Motion**: Page transitions, scroll reveals, interactive modals, and a custom magnetic cursor.
- **Lenis Smooth Scroll**: Buttery smooth scrolling experience.
- **Dynamic Projects**: Interactive project cards with deep links and technical metadata.
- **Fully Responsive**: Mobile-first design architecture.

## Getting Started

### Prerequisites
Make sure you have Node.js (version 18.x or later) installed.

### Installation
Clone the repository, then install dependencies:
```bash
npm install
# or
yarn install
```

### Running Locally
Start the development server:
```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the portfolio.

## Deployment on Vercel

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to a GitHub repository.
2. Go to Vercel and create a new project.
3. Import your GitHub repository.
4. Leave the default build settings as they are automatically optimized for Next.js (`npm run build`, output directory: `.next`).
5. Click **Deploy**.
